CHARACTER LIMIT RULES (≤300 chars):
LinkedIn notes have a 300 character limit. Always check the counter shown in the textarea.
Greeting rule:

Use "Hi [FirstName]," if the full message stays ≤300 chars
Use "Hi," if adding the name would push it over 300 chars
NEVER shorten or nickname names (e.g. do NOT use "Sam" for "Samantha")
NEVER send without a note — always click "Add a note"

How to calculate: Count base template chars (without greeting) first, then add len("Hi [Name], ") = name length + 5 chars.
