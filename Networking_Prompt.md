**1. Target Companies:**
Walmart, Oracle, Publicis Sapient, Providence, Innovaccer, Target, Lowe's, JPMC, Wells Fargo, BNY Mellon, American Express, Yellow.ai, Sarvam AI, Uniphore, Haptik, Keka, Zoho, greytHR, Postman, BrowserStack, Slice, Zepto, Swiggy, Zomato.

**2. Search & Filter Criteria:**
- **Job Titles:** "Graduate Trainee Engineer", "Associate Software Engineer", "SDE 1", "Backend Developer".
- **Target People:** Recruiters, Engineering Managers, or Senior Software Engineers.
- **Location Filter:** Hyderabad, Bangalore, or Chennai.

**3. Execution Steps:**
1. Search company + role on LinkedIn.
2. Filter by location (3 cities).
3. Connect + Add Note using the templates below.
